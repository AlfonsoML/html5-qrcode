  • resume(): void
    • Defined in html5-qrcode-scanner.ts:238

    Resumes the paused scan.

    If the video was previously paused by setting {@code shouldPauseVideo} to {@code true} in {@link Html5QrcodeScanner#pause(shouldPauseVideo)}, calling this method will resume the video.

    Notes:

    • Should only be called if camera scan is ongoing.

    throws
